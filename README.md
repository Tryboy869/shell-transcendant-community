## 💡 **AVANT vs APRÈS TRANSCENDANT**

### **❌ Shell Classique (Verbeux) :**
```bash
#!/bin/bash
set -euo pipefail
error_count=0
for file in *.log; do
    if [ -f "$file" ]; then
        if grep -q "ERROR" "$file"; then
            error_count=$((error_count + 1))
        fi
    fi
done
echo "Errors found: $error_count"
if [ $error_count -gt 5 ]; then
    echo "ALERT: Threshold exceeded"
fi
# 20+ lignes complexes !
```
